# Enterprise Network Lab - Cisco Packet Tracer

Simulación de infraestructura de red corporativa utilizando Cisco Packet Tracer. La topología incluye routers ISR, switches de capa 2/3 y Access Points, con segmentación VLAN, ACLs de seguridad y servicios básicos de red (DNS/DHCP).  

## Objetivos
- Configurar LAN/WAN segura y escalable
- Practicar segmentación de VLANs por departamentos
- Implementar políticas de seguridad ACL
- Integrar servicios corporativos: DHCP y DNS

## Tecnologías
- Cisco Packet Tracer
- Routers ISR 4321
- Switches 2960 & 3650
- Wireless Access Points
